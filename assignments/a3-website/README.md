
## Purpose
This website serves as a host for many of my projects that I have created throughout my first year in **Interactive Design & Technology.** This website utilizes **Bootstrap** extensively for CSS classes, while also having a bit of **GSAP** and **Hover.css** implementation.

## Bootstrap

[Bootstrap](https://getbootstrap.com/) is a library I used for essentially all styling across my portfolio page, except for a few inline adjustments. The headers and footers, cards, and carousels are all possible through Bootstrap classes, as well as the general colour palette and layout of the website. I used both Flexbox and Grid classes to achieve different layouts, and all pages are entirely responsive thanks to Bootstrap. For more details on the specific Bootstrap resources I used, please look at the comments on my pages.

## GSAP

[GSAP](https://gsap.com/) is a JavaScript animation library I used minorly, but to great effect. The most eye-catching feature when you first load the site is the scroll effect of my cover images, which was achieved using GSAP. As I develop this website more in the future, I plan to use GSAP more to create beautiful animations!

## Hover.css

[Hover.css](https://ianlunn.github.io/Hover/) is a CSS animation library for simple but clean hover effects. I used it for my cards and homepage carousel, to highlight clickable items and external links. I also used it for drop shadow hover effects for the cards on my semester pages.
