Reflection

This was an interesting and fun experiment into how libraries can improve user experience. JQuery and Animate.css are fascinating toolkits for me to experiment with more in the future. I found implementation with jQuery challenging initially, as I had to wrap my head around it's unique syntax. However, once I got the hang of it, it was surprisingly intuitive. I love the ease of selecting different HTML elements, and how it unlocks so many advanced features that base JavaScript is limited in. I found it more challenging to use in tandem with regular JavaScript, and I had to use AI for help in implementing it into my code correctly. With more time for experimenting and trial and error, I think I can implement jQuery in a variety of unique ways. I am excited to implement these ideas in the future!
When it comes to Animate.css, I found it much easier to use than any JS libraries I tried to implement. It was almost as easy as adding external fonts! The animations it adds are clean and classy, and I can see how they would be helpful in designing modern websites. I definitely want to explore it and other libraries, such as Hint.css, more in the future.
Overall, I enjoyed testing out these new libraries, and I'm excited about the features they bring to JavaScript! I would have liked to iterate more and flesh out my ideas, but I was limited in my time with a very busy schedule. For future projects, I will look into more libraries and continue to push the limits of my abilities within JavaScript, HTML, and CSS.
